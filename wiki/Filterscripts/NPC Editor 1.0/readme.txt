NPC editor 1.0
by 009 (Alex009)

Description:
Edit your NPC in game, good for videos.Use Controllable NPC plugin.Controll cmds:
> /npc add - create NPC
> /npc del <npcid> - destroy NPC
> /npc skin <npcid> - start select skin (Controll keys: left,right - select. walk - complete)
> /npc move <npcid> - start change position (Controll keys: up,down,left,right,sprint,crouch,analog left,analog right)
> /npc complete <npcid> - complete current edit
> /npc stop <npcid> - stop current action
> /npc animation <npcid> <animationid> <param1> <param2> <param3> <param4> <param5> - start select animation (Controll keys: left,right - select. walk - complete)
> /npc play <npcid> <playback name> - start play playback
> /npc rec start <playback name> - start recording playback (Controll keys: submission - stop rec)
> /npc rec stop - stop recording playback
